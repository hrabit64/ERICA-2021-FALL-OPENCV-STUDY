# ERICA 2021 가을학기 컴퓨터 비전 또래 멘토링



## Open CV를 활용한 컴퓨터 비전



한양대학교 에리카 캠퍼스에서 2021 가을학기 또래 맨토링입니다.



이 레포에서는 OpenCV를 활용한 컴퓨터 비전을 다루고 있으며,

모든 파일은 주피터 노트북으로 작성되어있습니다.



**OpenCV - 4.5.4** 버전을 사용중이며 사용하는 언어는 **파이썬** 입니다.
